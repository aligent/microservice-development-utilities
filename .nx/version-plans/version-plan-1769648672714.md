## create-workspace

### Features

- **New CLI tool for workspace creation**: Introduced `@aligent/create-workspace`, a command-line tool that streamlines the creation of Nx workspaces with opinionated presets
- **Interactive workspace setup**: Provides interactive prompts for preset selection and workspace naming with built-in validation
- **Preset support**: Supports custom Nx presets (e.g., `@aligent/nx-cdk`) with version pinning capability
- **Automatic cleanup**: Automatically removes unnecessary files after workspace creation
- **Flexible usage**: Can be used via npx, global installation, or with CLI arguments for automation
- **Comprehensive documentation**: Includes detailed README with usage examples, prerequisites, and troubleshooting

## nx-cdk

### Features

- **Migrated to @aligent/cdk-aspects**: Removed internal aspect implementations in favor of the dedicated `@aligent/cdk-aspects` package
