### New: remove-service generator

- Added a new generator to remove services from an NX CDK workspace, including spec tests.

### Improved: service generator

- Added an `--example` flag to optionally scaffold example service files (Step Functions definition, runtime handler, infra index).
- Refactored the base service template (index.ts.template) to a minimal scaffold.

### Improved: preset generator

- Added `--example` flag support to scaffold example infra code during workspace creation.
- Updated README, .gitignore, and application README templates.
- Simplified nxJson configuration and updated base package dependencies.

### Infra:

- Added `--access public` flag to the nx release publish command to publish workflow
- Renamed prefix from TEMPLATE-UPDATE to NX-CDK-TEMPLATE & NX-APPBUILDER-TEMPLATE in dependabot
