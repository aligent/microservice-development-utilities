## microservice-util-lib

- Add openapi-fetch middleware that supports:
  - Exponential backoff by default (linear and custom delay are supported)
  - Retry on idempotent HTTP methods by default (with option to disable)
  - Retry on network errors, 429 (Too Many Requests), 408 (Request Timeout) and 5XX (Server Errors)
  - Throw error on 4XX and 5XX status codes

## nx-openapi

- Fix import issue after client is generated by renaming client.ts.template to index.ts.template
- Pin openapi-typescript to version 7.7.3 to avoid transform errors
- Update tsconfig templates for better TypeScript support
