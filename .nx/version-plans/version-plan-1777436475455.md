nx-appbuilder: minor

- Nx plugin with two generators for Adobe App Builder development.
- `preset` generator scaffolds a new App Builder workspace (used by `create-nx-workspace --preset=@aligent/nx-appbuilder`).
- `app` generator scaffolds a new App Builder app inside an existing workspace, with feature flags for admin UI, business config, Commerce webhooks, events, REST actions, scheduled actions, and custom install steps.
