# @aligent/aws-wrappers

Opinionated AWS SDK wrappers with Powertools logging and X-Ray tracing

## Build

This library is written in TypeScript and can be built using:

```sh
npm install
npm run build
```

## Installation

Install this package via npm:

```sh
npm install --save @aligent/aws-wrappers
```

## Testing & Linting

Tests, linting & type-checking can be run with:

```sh
npm run test
```
